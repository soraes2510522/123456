피보나치 수열 알고리즘 탐구

1. 피보나치 수열이란?

피보나치 수열(Fibonacci Sequence)은 앞의 두 항의 합으로 다음 항이 만들어지는 수열이다. 예를 들어, 0, 1, 1, 2, 3, 5, 8, 13, … 순서로 나타난다.

2. 황금비(Golden Ratio)와의 관계

황금비는 인간이 인식하기에 가장 균형적이고 아름답다고 느끼는 비율이다. 피보나치 수열의 항들 사이의 비율은 점점 1.618에 가까워진다. 자연에서도 황금비를 찾아볼 수 있으며, 모나리자의 얼굴, 레오나르도 다빈치의 인체 비율, 해바라기 씨, 은하계 등에서 나타난다.

4. 황금비 수렴 과정

피보나치 수열의 항들 사이 비율 F(n)/F(n-1)은 n이 커질수록 점점 황금비 약 1.618에 가까워진다. 예를 들어, 처음 몇 항의 비율을 보면 1.000 → 2.000 → 1.500 → 1.667 → 1.600 … 와 같이 변하며, 항이 커질수록 1.618에 수렴한다.

이를 통해 수열이 단순한 숫자의 나열이 아니라, 자연과 예술 속 황금비와 연결된 규칙적인 구조임을 확인할 수 있다. 표와 그래프를 통해 직접 확인하면, 비율이 1.618로 점점 가까워지는 모습을 직관적으로 볼 수 있다.
