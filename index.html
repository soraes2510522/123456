피보나치 수열 알고리즘 탐구

① 알고리즘이란?

알고리즘이란 문제를 해결하기 위해 필요한 명확한 절차와 규칙의 집합이다. 컴퓨터는 스스로 생각하지 않고, 사람이 정해준 알고리즘을 순서대로 정확하게 실행한다.

본 프로그램은 반복문 기반 알고리즘을 이용하여 수열을 단계적으로 계산하는 과정을 보여준다.

② 피보나치 수열이란?

피보나치 수열은 앞의 두 항의 합으로 다음 항을 만드는 수열이다. 단순한 규칙이 반복되면서 빠르게 증가하는 특징을 가진다.

수학적으로는 다음과 같이 정의된다.

F(0) = 0, F(1) = 1
F(n) = F(n−1) + F(n−2)

③ 사용한 알고리즘 구조

이 프로그램은 이전 두 항의 값을 배열에 저장하고, 반복문을 사용하여 다음 항을 순차적으로 계산한다.

재귀 방식에 비해 같은 계산을 반복하지 않으므로 시간과 메모리 측면에서 효율적이다.

④ 피보나치 수열 계산

항의 개수를 입력하면 해당 개수만큼의 피보나치 수열을 계산하고, 계산 결과를 그래프로 시각화한다.

⑤ 피보나치 수열 증가 그래프

그래프의 x축은 항 번호 n, y축은 피보나치 수 F(n)을 나타낸다. 항이 증가할수록 값이 급격히 커지는 피보나치 수열의 특징을 시각적으로 확인할 수 있다.

⑥ 황금비와의 관계

황금비는 약 1.618로, 전체와 부분의 비율이 같아지는 조화로운 비율이다.

피보나치 수열에서 연속된 두 항의 비 F(n)/F(n−1)은 항의 개수가 증가할수록 황금비에 점점 가까워진다.
