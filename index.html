피보나치 수열 알고리즘 탐구

1. 피보나치 수열이란?

피보나치 수열(Fibonacci Sequence)은 앞의 두 항의 합으로 다음 항이 만들어지는 수열입니다.

수학적으로는 다음과 같이 정의됩니다:

F(0) = 0, F(1) = 1, F(n) = F(n-1) + F(n-2) (n ≥ 2)

2. 황금비와의 관계

황금비(φ)는 약 1.618이며, 피보나치 수열의 항들 사이 비율 F(n)/F(n-1)은 n이 커질수록 황금비에 가까워집니다. 자연과 예술 속에서도 발견됩니다.

황금비 공식: φ = (1 + √5) / 2
